# COVID-19 Data Analysis and Forecasting

## Overview
This project analyzes COVID-19 datasets using Python for data analysis, visualization, and forecasting.

## Features
- Exploratory Data Analysis (EDA)
- WHO regional breakdowns
- Mortality and recovery analysis
- Time-series forecasting
- Interactive visualizations

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Plotly
- Prophet

## Dataset
COVID-19 global datasets from WHO/Our World in Data.

## Visualizations
- Line graphs
- Bar charts
- Heatmaps
- World maps

## Project Goals
- Understand pandemic trends
- Analyze infection and recovery rates
- Predict future cases using forecasting models
